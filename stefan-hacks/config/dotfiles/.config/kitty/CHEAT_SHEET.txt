╟──────────────────────────────────────────────────────────────────────────────╢
║ • Use Ctrl+1-9 to jump directly to window (faster than Alt+H/J/K/L)         ║
║ • Ctrl+Shift+R for precise interactive resizing                             ║
║ • Ctrl+Shift+E resets all window sizes to equal                             ║
║ • Ctrl+Alt+Space rotates windows in current layout                          ║
║ • F11 toggles fullscreen for focused window                                 ║
║ • Broadcast mode (Ctrl+Shift+B) sends input to ALL windows                  ║
║ • Ctrl+Shift+H opens scrollback in Neovim (requires plugin)                 ║
║ • Use Stack layout for distraction-free full-screen focus                   ║

═══════════════════════════════════════════════════════════════════════════════
                             QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

1. INSTALL REQUIREMENTS
   • Kitty terminal
   • Hack Nerd Font
   • Catppuccin Mocha theme (download to ~/.config/kitty/current-theme.conf)

2. FIRST LAUNCH
   Ctrl+T           → Create new tab
   Alt+\            → Split vertically
   Alt+H/J/K/L      → Navigate between windows
   Ctrl+Shift+F     → Try different layouts

3. COMMON PATTERNS
   Code + Terminal:  Alt+\ then Ctrl+Shift+Alt+T (tall layout)
   Full focus:       Ctrl+Shift+Alt+S (stack layout)
   Monitoring:       Ctrl+Shift+Alt+G (grid layout)

4. CUSTOMIZE
   Ctrl+Shift+Alt+, → Edit config
   Ctrl+Alt+R       → Reload config
